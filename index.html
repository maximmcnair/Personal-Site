I'm a UI developer working in London on the js development of Fresh8's CMS. Below are a few side projects and prototypes I've done over the last couple of years. Interested in my work let's talk.

Crunchd

Crunchd is a network for fruit and veg growers to swap tips, and produce, with other green fingered folk in their area. I worked with Synth Media to created a Backbone/PhoneGap powered app for use on Andriod & IOS.

  • PhoneGap
  • Backbone JS
  • Android
  • IOS
  • Design
  • Grunt

See it in Person

This site pulls in photos from the Instagram API, locating their nearest airport from the photo’s lat/long. It then incorporates the SkyScanner API to deliver flight information to the user. Front end is built in React.js, allowing the initial view to being render server-side, perfect for SEO and page load time.

  • Node JS
  • MongoDB
  • React.js
  • Browserify

Potato and Mash

A recipe aggregator, using Cheerio.js to scape recipes from various sites. Which are then classified into categories using the Natural.js, a Natural Language Processing module for Node. Front end was kept simple letting Express.js render the views, with some ajax functionality for changing views.

  • Node JS
  • MongoDB
  • Design
  • Gulp

Talllies

Talllies is time tracking that forces you to work against a deadline, insipred by Parkison's Law, that "work expands so as to fill the time available for its completion". Built with a mix of Marionette enhanced Backbone on the front end with a REST API built with Node, Express & MongoDB.

  • Backbone JS
  • Marionette JS
  • Node JS
  • MongoDB
  • Design
  • Grunt
